See the following thinks -

1. Chrome driver is used in this script.
2. Chrome driver executable file are added on 'webdriver' folder which is located at the root of the project folder.
3. Chrome browser version must be 71-73.
4. The browser option is set as 'headless'. Just set the headless option as false in the ChromeDriverInitializer class file if headless is not wanted.
5. An test plan added on the 'TestPlan'folder which is located at the root of the project folder.
6. The whole script is followed with that 'TestPlan'.
7. After completing the script, the result of the test will show on a csv file which is located 'ResultFile'.
8. The result of the test will show the status of the test according with the test ID.
